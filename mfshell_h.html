🤔 Información y ayuda

Comandos para obtener información

/about

Muestra un breve resúmen acerca del bot y el enlace al sitio web propio del bot

Uso:

/about

/help

Muestra enlaces a la guía de usuario y a la referencia de comandos, es decir, este sitio

Uso:

/help

/info

Mostrar las propiedades de un archivo del lado de Telegram. Se usa respondiendo a un archivo

Use:

/info

/mediainfo

Información multimedia de un archivo (dentro del bot) obtenida al ejecutar MediaInfo

La información completa se le dará en un archivo de texto que se sube al chat, en caso de no subirlo (dios sabe por qué) el archivo se queda de todos modos en el directorio donde ejecuta el comando

Use:

/mediainfo A

A = Selección simple. Sólo un archivo

/start

Muestra información resumida sobre su sesión en el bot. Algunos datos no se muestran por esta vía

Uso:

/start

🤔 Sistema de archivos del bot (Básico)

Comandos básicos para navegar por el sistema de archivos del bot

/back

Cambiar de directorio al directorio contenedor o padre

Básicamente lo mismo que ejecutar: /cd -1

Uso:

/back

/cd

Cambiar de directorio

Puede ser un subdirectorio o algún nivel superior respecto a la ubicación donde usted se encuentra

Uso:

/cd A

A = Selección simple

/ls

Listar el contenido de una ubicación

Puede ser la ubicación actual, un subdirectorio, un archivo, varios elementos a la vez (rango) o algún nivel superior respecto a la ubicación donde usted se encuentra

Uso:

/ls A

A = Selección simple o por rango (Opcional)

/mkdir

Crea un subdirectorio nuevo

Uso:

/mkdir A

A = Nombre del subdirectorio

/mv

Este comando hace una de dos cosas: Mover uno o más elementos a una ubicación o renombrar un elemento

Use:

/mv A1 A2

Caso 1: Mover

A1 = Selección simple o de rango. Solamente archivos y subdirectorios

A2 = Ubicación. Puede ser un nivel superior

Caso 2: Renombrar

A1 = Selección simple. Un archivo o subdirectorio

A2 = Nuevo nombre

/rm

Borra uno o varios elementos. Mucho cuidado con esto porque no pide confirmación

Uso:

/rm A

A = Selección simple o de rango. Solamente archivos y subdirectorios

🤔 Sistema de archivos del bot (Avanzado)

Comandos para trabajar con archivos dentro del bot

/bren

Renombrado múltiple de archivos

Cuando ejecute el comando, se le va a mostrar una "simulación" de cómo van a quedar los archivos, y en ese mensaje sale para ejecutar el comando /ok para confirmar, puede ejecutar otro comando cualquiera para cancelar

Hay 3 operaciones: agregar, enumerar, remplazar. Cada operación se ejecuta según unos parámetros, los cuales tienen a "/" como separador

(A) Agregar:

Agrega una cadena de caracteres adicional al principio(prefijo) o al final (sufijo)

Formato (prefijo): Agregar/

Formato (sufijo): /Agregar

(E) Enumerar:

Renombra los archivos en el orden dado de forma tal que queden de la forma: NuevoNombre + Número + ExtensiónOriginal

Formato: NuevoTallo/NúmeroInicial/CantCifras

(R) Remplazar:

Remplaza en cada archivo seleccionado una cadena de caracteres con una nueva cadena de caracteres

Formato: Objetivo/Nuevo

Uso:

/bren A1 A2 A3

A1 = Selección simple, de rango o libre. Solamente de archivos

A2 = Tipo de operación: A (Agregar), E (Enumerar), R (Remplazar)

A3 = Parámetros de la operación

En este vídeo sale cómo se usa este comando para enumerar y remplazar

/crypt

Cifrado de archivos bajo la especificación Fernet

Antes de usar este comando, debe asegurarse de que KFSCrypt funcione en su PC y debe acostumbrarse a cifrar y descifrar archivos

KFSCrypt para Windows (32 bits)

KFSCrypt para Linux (64 bits)

Para cifrar o descifrar debe primero crear una clave ya sea ejecutando este comando sin argumentos o con KFSCrypt en su PC

Las claves generadas son completamente aleatorias y puede usarlas las veces que quiera

Para descifrar un archivo, debe hacerlo con la misma clave con el cual se cifró, en el caso de perder la clave ni se esfuerce porque ha perdido el archivo, es lo mismo que si se le olvidara la contraseña para un archivo comprimido

Cuando genera una clave con el comando, la clave se establece como la que se va a usar y el archivo con la clave se sube al chat

Uso:

/crypt A1 A2

Caso 1: Sin argumentos

Si se ejecuta respondiendo a un archivo en el chat, podrá ver y comparar claves

Si se ejecuta a secas, muestra la clave actual, si la tiene y los enlaces de descarga de KFSCrypt

Caso 2: Creación de una clave

A1 = "c" Crear una nueva clave

A2 = "f" Forzar creación de una nueva clave. Esto es en el caso de que ya haya una clave existente

Caso 3: Cualquier otro caso

A1 = Puede ser "r","e" o "d". El primero es para leer y asignar una clave a partir de un archivo del lado de Telegram. El segundo es para cifrar la selección de archivos hecha. El tercero es para descifrar la selección de archivos hecha

A2 = Selección simple de un archivo o un subdirectorio. En el caso de ser un subdirectorio, se seleccionan todos los archivos en el primer nivel

En este vídeo se muestra cómo usar KFSCrypt en Windows

/seven

Archivador 7Z sin compresión. Puede archivar un archivo o un directorio en un único volumen o múltiples partes y hasta puede ponerle protección con contraseña

Los archivos 7z pueden extraerse con 7Zipper (Android), 7zip (Windows). El autor no recomienda usar ni WinRAR, ni el RAR para Android

Uso:

/seven A1 A2 A3

Caso 1: Archivo único

A1 = Selección simple archivo o subdirectorio

A2 = Contraseña (Opcional)

Caso 2: Multivolumen

A1 = Selección simple archivo o subdirectorio

A2 = Tamaño de partes en MB (entero positivo, no se aceptan fracciones)

A3 = Contraseña (Opcional)

🤔 Control de colas

Comandos para controlar las colas

/brake

Frena o libera el freno de una cola

Las colas pueden frenarse sin que el proceso relacionado esté trabajando

Uso:

/brake A

A = Identificador de cola

/cancel

Cancelación de una cola

En las colas, el elemento cero (el primero) no se puede borrar, solamente se puede cancelar

Use:

/cancel A

A = Identificador de cola

/que

Si se ejecuta sin argumentos, se mostrarán en pantalla todas las colas y al final, un mensaje donde salen todos los identificadores de cola

Si se ejecuta con un argumento (un identificador de cola) es para ver una cola en específico

Si se ejecuta con dos argumentos es para borrar uno o varios elementos de una cola

El elemento cero de la cola no se puede borrar, para ello debe usar /cancel. Si en una selección por rango selecciona el elemento cero, este será ignorado y si selecciona el elemento cero en una selección simple, se borrarán todos los elementos de la cola excepto el elemento cero

Use:

/que A1 A2

A1 = Identificador de cola (Opcional)

A2 = Selección simple o de rango para borrar de la cola (Opcional)

🤔 Transferencias de red (Telegram)

Comandos para las transferencias de red entre el bot y Telegram

/auto

Activar o desactivar la descarga automática de los archivos que son reenviados hacia el chat. Desactivado de forma predeterminada. Es la única forma de descargar álbumes/grupos de archivos

Uso:

/auto

/download

Descarga un archivo de Telegram al bot. Se usa respondiendo a un archivo. Opcionalmente puede cambiar el nombre del archivo al descargarlo

o:

/download A

A = Nuevo nombre (Opcional)

/upload

Sube archivos del bot a Telegram, puede ser uno o varios archivos

Tenga en cuenta que no se pueden subir archivos que superen 2 GB de espacio, ni superen 60 caracteres en el nombre, si los archivos no se encolan para subir, debe revisar y asegurarse de que cumplan con estos requisitos

Uso:

/upload A

A = Selección simple (archivos y directorios) o de rango (solamente archivos). En el caso de ser una selección simple sobre un directorio, se seleccionan todos los archivos en el 1er nivel de profundidad de ese directorio

🤔 Transferencias de red (Web)

Comandos de transferencias de red entre el bot y el resto de Internet

/gddl

Descargador de URLs de Google Drive

Uso:

/gddl A1 A2

A1 = Lista de URLs separadas por espacio

A2 = "R" (literalmente) para indicar en caso de si las URLs son carpetas de Google Drive, descargarlas de forma recursiva. Esto es opcional

/megacreds

Credenciales de cuenta de MEGA

Para usar una cuenta real de MEGA en lugar de depender de la sesión anónima

Uso:

/megacreds A1 A2

A1 = Usuario de MEGA

A2 = Contraseña del usuario de MEGA

/megadl

Descargador de enlaces MEGA. No soporta carpetas. Se usa una sesión anónima de forma predeterminada en caso de no haber credenciales de MEGA asignados

Use:

/megadl A

A = URLs de archivos en MEGA

/share

Compartir todos los archivos. Si activa esto, los archivos serán compartidos via web de forma pública para poder ser descargados desde el servidor web adjunto al bot. Desactivado de forma predeterminada

Puede ver en todo momento su enlace ejecutando /start

Uso:

/share

/ua

Agente de usuario

Afecta el comportamiento de /wget

Ejecute sin argumentos para ver el agente de usuario actual. De forma predeterminada, es aleatorio, pero puede elegir entre Chrome (c), Firefox (f), Safari (s), Opera (o), o aleatorio (r)

Uso:

/ua A

A = Agente de usuario (Opcional)

/wget

Descargador web general. Descarga enlaces directos (no todos) y es compatible con varios sitios de alojamiento de archivos, ejecute sin argumentos para ver los sitios soportados

Use:

/wget A

A = URLs separadas por espacios (Opcional)

/ytdl

Descargador de Youtube

Uso:

/ytdl A1 A2

A1 = URLs de vídeos de Youtube o listas de reproducción de Youtube separadas por espacios

A2 = Especificación. Puede ser "360p","720p","1080p" o "exa". Las 3 primeras son resoluciones de video, la última es para extraer audio. Todo esto es opcional
